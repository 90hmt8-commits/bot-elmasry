# Bot Elmasry

Bot Elmasry is a powerful bot designed to automate various tasks and enhance productivity in your daily workflows.

## Features
- **Task Automation:** Automates repetitive tasks to save time and effort.
- **Custom Commands:** Users can create and configure custom commands tailored to their needs.
- **User-Friendly Interface:** Easy to use without extensive technical knowledge.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/90hmt8-commits/bot-elmasry.git
   ```
2. Navigate into the project directory:
   ```bash
   cd bot-elmasry
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Launch the bot:
   ```bash
   npm start
   ```

## Commands
- **!help:** Displays a list of available commands and their usage.
- **!setup:** Initializes the bot with the necessary configurations.
- **!status:** Checks the current status of the bot.
- **!custom [command]:** Adds or modifies a custom command.
